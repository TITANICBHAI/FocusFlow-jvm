데스크톱 잠금.
깊은 집중.

FocusFlow는 Win32를 통해 차단된 프로세스를 강제 종료하고, 집중 런처로 Windows 데스크톱을 방해 없는 키오스크 모드로 잠그며, PIN 게이트로 집중 세션을 강제합니다. 소프트 타이머 없음. 우회 방법 없음. 실제 적용.

집중 런처

전체 데스크톱 잠금.
PIN 없이는 탈출 불가.

집중 런처는 인지행동치료에서 영감을 받은 키오스크 모드로, 세션이 시작되는 순간 Windows 데스크톱을 방해 없는 작업 공간으로 대체합니다.

신규 기능

키오스크 데스크톱 모드

세션이 시작되는 즉시 Windows 작업 표시줄을 숨기고 핵폭탄 모드를 활성화합니다. 빠른 알림 확인 없음. 작업 표시줄 방해 없음. 오직 당신의 일만.

  • 세션 전체에서 작업 표시줄 숨김
  • 세션 시작 시 핵폭탄 모드 자동 활성화
  • 전체 화면 집중 오버레이가 데스크톱 덮음
약속 장치

PIN 게이트 휴식

모든 휴식에는 세션 PIN이 필요합니다. 하드 잠금 모드는 휴식을 완전히 건너뜁니다. 충동과 행동 사이의 인지행동치료 기반 마찰이 '잠깐만 확인할게'와 실제로 집중을 유지하는 것의 차이를 만듭니다.

  • 5분 PIN 게이트 휴식
  • 하드 잠금 모드 — 휴식 없음
  • SHA-256 PIN — 우회 불가능

충돌 안전 설계

FocusFlow가 충돌해도 작업 표시줄이 복원됩니다. 세 개의 독립적인 안전 레이어가 종료 시 실행됩니다 — 시작 복원, JVM 종료 훅, 전역 충돌 핸들러.

안전 입력 화이트리스트

55개 이상의 시스템 프로세스가 항상 보호됩니다 — 키보드 드라이버, OEM 터치패드 소프트웨어, 마우스 주변기기, 오디오, UAC 및 접근성 도구. FocusFlow는 입력 장치를 절대 손상시키지 않습니다.

실제 적용

프로세스 종료. 방화벽 규칙 추가.
시도 기록됨.

모든 적용 작업은 두 가지 독립적인 감지 방법을 사용하여 즉시 실행됩니다 — 유예 기간 없음, 경고 대화상자 없음, 미루기 버튼 없음.

WinEventHook을 통한 즉시 감지

차단된 앱이 포그라운드로 오는 순간 SetWinEventHook이 실행됩니다. 500ms 폴링 루프가 훅이 놓친 것을 잡아냅니다 (UWP 앱 포함).

Win32/JNA를 통한 프로세스 종료

ProcessHandle.destroyForcibly()가 먼저 실행됩니다. taskkill /F /IM 폴백이 종료를 보장합니다.

Windows 방화벽 규칙

네트워크 차단은 차단된 앱에 대한 실시간 아웃바운드 netsh advfirewall 규칙을 추가합니다 — Windows 방화벽 레이어에서 시행됩니다.

유혹 일기

차단된 모든 시도가 로컬 SQLite 데이터베이스에 기록됩니다. 차트로 시간에 따른 방해 패턴을 확인하여 습관을 이해하고 변화시킬 수 있습니다.

웹사이트 차단기

hosts 파일 레벨에서 도메인 차단 — 모든 브라우저에 동시 적용. 확장 프로그램 불필요.

키워드 차단기

제목에 키워드가 포함된 모든 창 차단 — 모든 도메인을 나열하지 않고 SNS 탭, 뉴스 사이트, 채팅 앱 캐치.

핵폭탄 모드

최대 적용 — 작업 관리자, PowerShell, 실행 대화상자, CMD 및 30개 이상의 다른 탈출 경로 차단.

차단 일정

앱이 자동으로 차단되는 반복 시간대 설정 — 예: 평일 오전 9시~오후 5시 SNS 차단.

일일 허용량

앱에 일일 시간 예산 설정 — 예: YouTube 하루 30분. 예산이 다 소진되면 당일 남은 시간 차단.

차단 방어

세션 중간에 자신의 차단 규칙을 비활성화하기 얼마나 어렵게 할지 설정. 쉬움, PIN 필요 또는 세션 종료까지 잠금.

집중 세션

뽀모도로 또는 사용자 정의 길이 세션, PIN 게이트, 휴식 강제 기능, 차단된 앱을 열려고 할 때마다 울리는 소리 혐오 음.

통계 & 보고서

집중 연속 기록, 세션 이력, 유혹 일기 차트, 매주 일요일 자동 전송되는 주간 집중 보고서.

습관 트래커

7일 연속 기록 그리드로 일일 습관 형성. 집중 세션과 연동된 습관으로 가장 생산적인 날을 이끄는 규율 확인.

작업 관리

우선순위, 마감일, 반복 규칙(매일/평일/매주/매월), Windows 알람 알림이 포함된 완전한 작업 CRUD.

일일 메모

세션 데이터와 연결된 개인 일기. 세션 전에 하루를 계획하고 세션 후에 검토 — 모두 로컬 저장, 동기화 없음.

100% 프라이버시

계정 없음. 인터넷 불필요. 분석 없음. 자동 롤링 백업이 있는 로컬 SQLite 데이터베이스에 모든 데이터 저장. 앱 제거, 폴더 삭제 — 완전히 사라짐.

FAQ

자주 묻는 질문

FocusFlow는 무료인가요?
예 — Microsoft Store에서 완전히 무료입니다. 구독, 앱 내 구매, 계정 등록이 필요하지 않습니다.
FocusFlow는 Windows 11에서 작동하나요?
예. FocusFlow는 Windows 10 및 Windows 11과 완전히 호환됩니다.
포커스 런처란 무엇인가요?
포커스 런처는 FocusFlow의 키오스크 모드입니다. 선택한 앱만으로 Windows 데스크탑을 잠그고 작업 표시줄을 숨기며 다른 모든 프로세스를 차단합니다.
FocusFlow는 어떻게 앱을 차단하나요?
FocusFlow는 Win32 API를 사용해 차단된 프로세스를 실시간으로 종료합니다. 소프트 타이머가 없고 우회할 방법도 없습니다.
문제가 생기면 작업 표시줄을 복구할 수 있나요?
예. FocusFlow에는 세 가지 독립적인 복구 레이어가 있어 작업 표시줄이 항상 복원됩니다.
FocusFlow가 웹사이트를 차단하나요?
예. Windows hosts 파일 수준에서 차단하므로 확장 프로그램 없이 모든 브라우저에서 동시에 작동합니다.
