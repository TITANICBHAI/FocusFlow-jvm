Características

Todo lo que hace FocusFlow

Un desglose completo de cada herramienta en la pila de aplicación de FocusFlow.

Núcleo

Terminación real de procesos

FocusFlow termina las aplicaciones bloqueadas mediante llamadas a la API Win32, no temporizadores suaves.

Modo Quiosco

Focus Launcher

Bloquea todo el escritorio de Windows a las aplicaciones elegidas. La barra de tareas se oculta y todos los demás procesos se terminan.

Bloqueo

Modo Nuclear

Desactiva Win+D, Win+Tab, el Administrador de tareas y todos los atajos del sistema durante una sesión de enfoque.

Bloqueo web

Bloqueador de hosts

Bloquea sitios web a nivel del archivo hosts de Windows — funciona en todos los navegadores sin extensión.

Red de seguridad

Kill Switch

Presupuesto de emergencia diario de 5 minutos para pausar toda la aplicación. Requiere PIN.

Seguridad ante fallos

Sistema de recuperación triple

Restauración al inicio, hook de apagado de JVM y manejador de fallos global restauran tu barra de tareas de forma independiente.

Diario

Notas diarias

Diario privado de sesiones para planificar y revisar tus sesiones de enfoque. Todo se almacena localmente.

Progreso

Hábitos y estadísticas

Rastrea rachas, recuentos de sesiones y tiempo de enfoque diario para ver patrones de productividad a largo plazo.
