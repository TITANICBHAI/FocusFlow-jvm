2. What FocusFlow Does

FocusFlow for Windows provides:

  • Focus Launcher — a CBT-style kiosk mode that hides the Windows taskbar, activates Nuclear Mode, and enforces PIN-gated breaks or hard-lock sessions for deep work
  • App blocking — real-time termination of user-selected Windows processes via Win32 APIs during focus sessions or always-on enforcement
  • Website blocking — hosts-file domain blocking affecting all browsers
  • Network blocking — Windows Firewall outbound rules via netsh advfirewall (requires administrator privileges)
  • Nuclear Mode — maximum enforcement mode that blocks over 30 system utility escape routes
  • Block Defense, Block Schedules, Daily Allowances, Keyword Blocker, Standalone Blocks
  • Focus session timers (Pomodoro and custom), session PIN gate, break enforcer, sound aversion
  • Task management with recurring tasks and alarm notifications
  • Habit tracker, daily notes, stats, reports, and focus insights
  • System tray integration and optional Windows startup via registry

All enforcement is performed locally on your Windows device and is subject to OS, hardware, software, and privilege-level limitations beyond FocusFlow's control.
