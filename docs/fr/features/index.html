Fonctionnalités

Tout ce que fait FocusFlow

Un aperçu complet de chaque outil dans la pile d'application de FocusFlow.

Cœur

Terminaison réelle des processus

FocusFlow termine les applications bloquées via des appels d'API Win32 — pas de minuteries douces. Aucun contournement n'existe.

Mode Kiosque

Focus Launcher

Verrouille tout votre bureau Windows sur les applications choisies. La barre des tâches disparaît et tous les autres processus sont terminés.

Verrouillage

Mode Nucléaire

Désactive Win+D, Win+Tab, le Gestionnaire des tâches et tous les raccourcis système pendant une session de concentration.

Blocage web

Bloqueur de fichier hosts

Bloque les sites web au niveau du fichier hosts Windows — fonctionne dans tous les navigateurs sans extension.

Filet de sécurité

Kill Switch

Budget d'urgence quotidien de 5 minutes pour mettre en pause toute l'application. Nécessite un PIN.

Sécurité

Système de récupération triple

Restauration au démarrage, hook d'arrêt JVM et gestionnaire de crash global restaurent votre barre des tâches indépendamment.

Journal

Notes quotidiennes

Journal de session privé pour planifier et réviser vos sessions de concentration. Tout stocké localement.

Progrès

Habitudes & Statistiques

Suivez les séries, le nombre de sessions et le temps de concentration quotidien pour voir les tendances de productivité à long terme.
