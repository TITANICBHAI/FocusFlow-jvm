Lanceur de Concentration

Verrouillez tout votre bureau.
Pas de sortie sans votre PIN.

Le Lanceur de Concentration est un mode kiosque inspiré de la TCC qui remplace votre bureau Windows par un espace de travail sans distraction dès qu'une session commence.

Nouvelle Fonctionnalité

Mode Kiosque Bureau

Masque la barre des tâches Windows et active le Mode Nucléaire dès le début de votre session. Pas de notifications rapides. Pas de distractions. Juste votre travail.

  • Barre des tâches masquée pendant toute la session
  • Mode Nucléaire activé automatiquement au démarrage
  • Overlay plein écran couvre le bureau
Dispositif d'Engagement

Pauses avec Code PIN

Chaque pause nécessite votre code PIN de session. Le mode Verrouillage Total supprime entièrement les pauses. Cette friction basée sur la TCC entre l'impulsion et l'action fait toute la différence.

  • Pauses de 5 minutes avec code PIN
  • Mode Verrouillage Total — aucune pause
  • PIN SHA-256 — ne peut pas être contourné

Conception Sécurisée en Cas de Crash

Votre barre des tâches sera restaurée même si FocusFlow plante. Trois couches de sécurité indépendantes se déclenchent à la fermeture — restauration au démarrage, hook d'arrêt JVM et gestionnaire de crash global.

Liste Blanche d'Entrée Sécurisée

Plus de 55 processus système sont toujours protégés — pilotes clavier, logiciels tactiles OEM, périphériques souris, audio, UAC et outils d'accessibilité. FocusFlow ne brise jamais vos périphériques d'entrée.

Application Réelle

Processus tué. Règle pare-feu ajoutée.
Tentative enregistrée.

Chaque action d'application se déclenche immédiatement via deux méthodes de détection indépendantes — sans période de grâce, sans boîte de dialogue d'avertissement, sans bouton de rappel.

Détection Instantanée via WinEventHook

Un SetWinEventHook se déclenche dès qu'une application bloquée passe au premier plan. Une boucle d'interrogation de 500 ms capture ce que le hook manque, y compris les applications UWP.

Suppression de Processus via Win32/JNA

ProcessHandle.destroyForcibly() se déclenche en premier. Un repli taskkill /F /IM garantit le succès de la suppression.

Règles Pare-feu Windows

Le blocage réseau ajoute une règle de sortie active netsh advfirewall — appliquée au niveau du pare-feu Windows.

Journal de Tentation

Chaque tentative bloquée est écrite dans une base de données SQLite locale. Des graphiques montrent vos patterns de distraction au fil du temps pour vous aider à comprendre et changer vos habitudes.

Bloqueur de Sites Web

Bloque les domaines au niveau du fichier hosts — fonctionne pour tous les navigateurs simultanément. Aucune extension nécessaire.

Bloqueur de Mots-clés

Bloque toute fenêtre dont le titre contient un mot-clé — capture les onglets de réseaux sociaux, sites d'actualités et applications de chat.

Mode Nucléaire

Application maximale — bloque le Gestionnaire des tâches, PowerShell, la boîte d'exécution, CMD et plus de 30 autres voies d'échappement.

Planifications de Blocage

Définissez des plages horaires récurrentes où les applications sont automatiquement bloquées — ex. réseaux sociaux bloqués de 9h à 17h en semaine.

Allocations Quotidiennes

Donnez un budget de temps quotidien à n'importe quelle application — ex. 30 minutes de YouTube par jour. Une fois le budget épuisé, elle est bloquée pour le reste de la journée.

Défense de Blocage

Configurez la difficulté à désactiver vos propres règles de blocage en cours de session. Facile, avec PIN ou verrouillé jusqu'à la fin de la session.

Suivi des Habitudes

FAQ

Foire aux questions

FocusFlow est-il gratuit?
Oui — entièrement gratuit sur le Microsoft Store. Pas d'abonnements, pas d'achats intégrés, aucun compte requis.
FocusFlow fonctionne-t-il sur Windows 11?
Oui. FocusFlow est entièrement compatible avec Windows 10 et Windows 11.
Qu'est-ce que Focus Launcher?
Focus Launcher est le mode kiosque de FocusFlow. Il verrouille tout votre bureau Windows sur les applications choisies, masque la barre des tâches et bloque tous les autres processus.
Comment FocusFlow bloque-t-il les applications?
FocusFlow utilise les API Win32 pour terminer les processus bloqués en temps réel. Il n'y a pas de minuterie douce ni de contournement possible.
Puis-je récupérer ma barre des tâches en cas de problème?
Oui. FocusFlow a trois couches de récupération indépendantes — votre barre des tâches est toujours restaurée.
FocusFlow bloque-t-il les sites web?
Oui. FocusFlow bloque les sites web au niveau du fichier hosts Windows — fonctionne dans tous les navigateurs sans extension.
