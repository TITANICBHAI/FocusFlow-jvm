专注启动器是一种受认知行为疗法启发的信息亭模式，会在会话开始时将您的 Windows 桌面替换为无干扰的工作空间。

全新功能

信息亭桌面模式

会话开始的瞬间隐藏 Windows 任务栏并激活核弹模式。没有快速查看通知的机会，没有任务栏干扰，只有您的工作。

  • 整个会话期间任务栏隐藏
  • 会话开始时自动启用核弹模式
  • 全屏专注覆盖层覆盖桌面
承诺装置

PIN 码验证休息

每次休息都需要您的会话 PIN 码。硬锁定模式完全跳过休息。这种基于认知行为疗法的冲动与行动之间的摩擦，正是"随便看一眼"与真正保持专注之间的区别。

  • 5 分钟 PIN 码验证休息
  • 硬锁定模式 — 完全无休息
  • SHA-256 PIN — 无法绕过

崩溃安全设计

即使 FocusFlow 崩溃，您的任务栏也会恢复。三个独立的安全层在退出时触发 — 启动恢复、JVM 关闭钩子和全局崩溃处理程序。

安全输入白名单

55 个以上的系统进程始终受到保护 — 键盘驱动程序、OEM 触摸板软件、鼠标外设、音频、UAC 和辅助功能工具。FocusFlow 绝不会损坏您的输入设备。

真实执行

进程已终止。防火墙规则已添加。
尝试已记录。

每项执行操作都通过两种独立的检测方法立即触发 — 无宽限期，无警告对话框，无暂停按钮。

通过 WinEventHook 即时检测

当任何被拦截的应用出现在前台时，SetWinEventHook 立即触发。500ms 轮询循环捕获钩子遗漏的内容，包括 UWP 应用程序。

通过 Win32/JNA 终止进程

ProcessHandle.destroyForcibly() 首先触发。taskkill /F /IM 后备方案确保终止成功。

Windows 防火墙规则

网络拦截为被拦截的应用添加实时出站 netsh advfirewall 规则 — 在 Windows 防火墙层执行。

诱惑日记

每次被拦截的尝试都写入本地 SQLite 数据库。图表显示您随时间的分心规律，帮助您理解并改变习惯。

网站拦截器

在 hosts 文件级别拦截域名 — 同时对所有浏览器生效。无需安装扩展。

关键词拦截器

拦截标题包含关键词的任何窗口 — 无需列出每个域名即可捕获社交媒体标签页、新闻网站和聊天应用。

核弹模式

最强执行 — 拦截任务管理器、PowerShell、运行对话框、CMD 和 30 多个其他逃生路线。

拦截计划

设置应用自动被拦截的循环时间窗口 — 例如工作日上午 9 点至下午 5 点拦截社交媒体。

每日配额

为任何应用设置每日时间预算 — 例如每天最多使用 30 分钟 YouTube。预算用尽后，当天剩余时间内将被拦截。

拦截防御

配置在会话中途停用自己的拦截规则的难度。可设为简单、需要 PIN 或锁定至会话结束。
