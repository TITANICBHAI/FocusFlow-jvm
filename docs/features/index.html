Features

Everything FocusFlow Does

A complete breakdown of every tool in FocusFlow's enforcement stack.

Core

Real Process Killing

FocusFlow terminates blocked apps via Win32 API calls — not soft timers. The moment a blocked process starts, it's ended. No workaround exists.

Kiosk Mode

Focus Launcher

Lock your entire Windows desktop to only the apps you choose. The taskbar hides, shortcuts disable, and all other processes are killed. One 5-minute break per day.

Lockdown

Nuclear Mode

Disables Win+D, Win+Tab, Task Manager, and all system shortcuts during a focus session. No escaping to the desktop mid-session.

Website Blocking

Hosts File Blocker

Blocks websites at the Windows hosts file level. Works in every browser — Chrome, Firefox, Edge — simultaneously, with no extension needed.

Safety Net

Kill Switch

Emergency 5-minute daily budget to pause all enforcement. Requires PIN to activate. Budget tracked per calendar day and persisted across restarts.

Crash Safety

Triple Recovery System

A startup restore, JVM shutdown hook, and global crash handler all independently restore your taskbar — even if FocusFlow crashes mid-session.

Journaling

Daily Notes

A private session journal to plan before a focus session and review after. All data stored locally — nothing is synced or uploaded anywhere.

Progress

Habits & Stats

Track streaks, session counts, and daily focus time. Persistent stats across sessions help you see long-term productivity patterns.
