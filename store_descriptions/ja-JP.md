## 説明（長文）
> **Partner Center → Description**（10,000 文字以内）

```
FocusFlow — Windows 用 深い集中 & アプリブロッカー

注意力を取り戻しましょう。FocusFlow は Windows 向けの最強の集中・生産性ツールです。「集中しよう」と提案するだけでなく、強制的に集中させます。プロセスレベルで気を散らすアプリをブロックし、デスクトップを集中作業空間に変え、生産的な習慣を定着させます。

学生、開発者、ライター、デザイナー、あるいは日々の誘惑と戦うすべての人へ。FocusFlow はあなたが毎日ベストを尽くすためのツールです。

──────────────────────────────────────
🔒 本物のアプリブロック — 通知ではなく強制終了
──────────────────────────────────────
FocusFlow は実行中の Windows プロセスを監視し、ブロック対象のアプリが起動した瞬間に閉じます。ブラウザタブだけでなく、実際のアプリケーションを対象とします。Chrome、Discord、Steam、Spotify など、任意の実行ファイルをブロック可能です。

• プロセス名で任意のアプリをブロック
• 常時ブロック、スケジュールブロック、またはセッション中のみのブロック
• 1日の許容時間設定 — アプリに時間予算を設定（例：1日 30 分の YouTube）
• Web サイトブロッカー — hosts ファイルレベルでドメインをブロック（全ブラウザ対応）
• キーワードブロッカー — 指定した語句がタイトルに含まれるウィンドウをブロック

──────────────────────────────────────
🖥 フォーカスランチャー — 深い作業のためのキオスクモード
──────────────────────────────────────
フォーカスランチャーはセッション開始と同時にデスクトップ全体を集中作業空間に変えます：

• Windows タスクバーを非表示 — 「ちょっと確認するだけ」がなくなります
• ニュークリアモードを自動起動 — 最初の1秒から最大限のブロック
• PIN 保護ブレーク — 休憩のたびにセッション PIN の入力が必要（衝動的な離脱を防止）
• ハードロックモード — 休憩なし、タイマー終了まで純粋な深い作業
• クラッシュセーフ設計 — アプリが予期せず終了してもタスクバーは必ず復元されます

──────────────────────────────────────
⏱ 集中セッションとポモドーロタイマー
──────────────────────────────────────
• 計測セッション：15・25・45・60・90 分、またはカスタム時間
• ポモドーロモード — 短い休憩と長い休憩を含む自動の作業/休憩サイクル
• セッション PIN — 早期終了できないように自分をロック
• タスクバーにリアルタイムカウントダウンを表示
• 音による嫌悪刺激 — ブロック中のアプリを開こうとするたびにアラート音

──────────────────────────────────────
☢ ニュークリアモード — 最強の強制力
──────────────────────────────────────
逃げ道をゼロにしたいときのニュークリアモードは、タスクマネージャー、PowerShell、ファイル名を指定して実行、ブラウザ開発者コンソールなど 30 以上のシステムツールをブロックします。集中すると決めたなら、ニュークリアモードがその決断を守ります。

──────────────────────────────────────
📊 統計・レポート・インサイト
──────────────────────────────────────
• 集中ダッシュボード — 連続日数、完了タスク、ブロック試行回数、集中スコア
• 週次レポート — 毎週日曜日に配信、ピーク時間と最高の作業日を確認
• 誘惑ログ — ブロックされた試行をタイムスタンプとアプリ名付きで記録
• セッション履歴 — 全セッションをローカルに保存（時間と結果付き）
• 自分のパターンに基づく生産性トレンドと改善提案

──────────────────────────────────────
✅ タスク・習慣・メモ
──────────────────────────────────────
• 時間帯・所要時間・優先度付きでタスクをスケジュール
• 繰り返しタスク — 毎日・平日・毎週
• 完了ストリーク付きの毎日の習慣トラッカー
• セッションメモと日次ログ — すべてローカルに保存

──────────────────────────────────────
🔐 完全プライベート — データはあなたの PC を離れません
──────────────────────────────────────
• アカウント不要 — インストールしてすぐ使える
• インターネット接続不要 — すべての機能が完全オフラインで動作
• テレメトリ・分析・使用データの送信は一切なし
• 全データは %USERPROFILE%\.focusflow\ 内の SQLite データベースにローカル保存
• 自動ローリングバックアップとクラッシュログはローカルのみ

──────────────────────────────────────
⚙ システムと利便性
──────────────────────────────────────
• システムトレイでサイレント動作 — 右クリックで復元・ブロック切替・終了
• Windows 起動時の自動起動（任意設定、管理者権限不要）
• セッション・ブロックイベント・週次レポートの Windows 通知
• Windows 10 および Windows 11 対応

──────────────────────────────────────
こんな方に FocusFlow
──────────────────────────────────────
• 勉強中に SNS を断ちたい学生
• 深い作業が必要な開発者・ライター・デザイナー
• 任意のタイマーではなく強制力のあるツールが必要な ADHD の方
• 他の集中アプリをすぐに閉じてしまった経験がある方

「ちょっとだけ見る」と思ったら1時間が過ぎていた経験があるなら — FocusFlow はあなたのためのツールです。
```
