## 説明（長文）
> **Partner Center → Description**（10,000 文字以内）

```
FocusFlow は Windows 向けの深い集中とアプリブロックツールです。注意力を取り戻し、生産的な習慣を築き、本当に重要な作業を完成させるお手伝いをします。

ソーシャルメディア、ゲーム、または単に自分への責任感に悩んでいる方でも、FocusFlow は優しいリマインダーではなく、本物の強制力を提供します。

──────────────────────────────────────
🔒 本物のアプリブロック
──────────────────────────────────────
FocusFlow は実行中のプロセスを監視し、ブロックされたアプリが開いた瞬間に閉じます。ブラウザタブではなく — 実際の Windows アプリケーションです。Chrome、Discord、Steam、Spotify、またはお好みの実行ファイルをブロックできます。

常時有効、特定の時間帯にスケジュール、またはアクティブな集中セッションに連動したブロックを設定できます。1日の許容時間設定により、自動的にブロックが始まる前にアプリごとに限られた時間を与えることができます。

──────────────────────────────────────
⏱ 集中セッションとポモドーロ
──────────────────────────────────────
任意のタスクに対して計測された集中セッションを開始できます。時間（15、25、45、60、または 90 分）を選択するか、カスタム時間を入力してください。ポモドーロモードを有効にすると、短い休憩と長い休憩を含む自動の作業/休憩サイクルが設定されます。

大きな円形タイマーでセッションを常に可視化。PIN ロックで早期終了を防ぎながら、一時停止、再開、または終了できます。残り時間を常に把握できるよう、ライブカウントダウンがタスクバーのタイトルに表示されます。

──────────────────────────────────────
📊 ダッシュボードとインサイト
──────────────────────────────────────
毎日のダッシュボードでは、集中のストリーク、完了したタスク、ブロックされたアプリの試行回数、集中スコアが表示されます。週次レポートはあなたのパターンをまとめます — ピーク生産時間、最高の日、平均セッション時間、合計集中時間。

──────────────────────────────────────
✅ タスク・習慣・メモ
──────────────────────────────────────
時間帯、時間、優先度でタスクをスケジュール。繰り返しタスク（毎日、平日、毎週）を設定。完了ストリーク付きの毎日の習慣を追跡。セッションメモと日次ログを保存 — すべてローカルに保存されます。

──────────────────────────────────────
🛡 ブロックディフェンスとニュークリアモード
──────────────────────────────────────
ブロックディフェンスは回避を防ぎます — 上書きしたい誘惑があっても強制力を守ります。ニュークリアモードは逃げ道のない最大限の集中のためにすべてをロックします。

──────────────────────────────────────
🔐 設計によるプライバシー保護
──────────────────────────────────────
すべてはデバイス上に留まります。アカウント不要。クラウド同期なし。データ収集なし。すべての設定、タスク、統計情報は PC 上の SQLite データベースにローカル保存されます。

──────────────────────────────────────
システムトレイで静かに動作します。ウィンドウが閉じられてもブロックはアクティブのままです。Windows 10 および Windows 11 向けに構築されています。
```
