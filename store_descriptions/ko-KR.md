## 긴 설명
> **Partner Center → Description** (10,000자 이내)

```
FocusFlow — Windows용 깊은 집중 & 앱 차단 도구

집중력을 되찾으세요. FocusFlow는 Windows에서 가장 강력한 집중·생산성 도구입니다. 단순히 집중하라고 권유하는 게 아니라, 강제로 집중하게 만듭니다. 프로세스 수준에서 방해 앱을 차단하고, 데스크톱을 방해 없는 작업 공간으로 잠그며, 지속적인 생산적 습관을 형성할 수 있도록 도와줍니다.

학생, 개발자, 작가, 디자이너, 또는 일상의 방해 요소와 싸우는 누구에게나 — FocusFlow는 매일 최선을 다할 수 있는 도구를 제공합니다.

──────────────────────────────────────
🔒 진짜 앱 차단 — 알림이 아닌 강제 종료
──────────────────────────────────────
FocusFlow는 실행 중인 Windows 프로세스를 모니터링하고, 차단된 앱이 실행되는 순간 즉시 닫습니다. 브라우저 탭이 아닌 실제 애플리케이션입니다. Chrome, Discord, Steam, Spotify 또는 원하는 모든 실행 파일을 차단할 수 있습니다.

• 프로세스 이름으로 모든 앱 차단
• 상시 차단, 예약 차단, 또는 세션 중에만 차단
• 일일 허용량 설정 — 앱에 시간 예산 부여 (예: 하루 YouTube 30분)
• 웹사이트 차단기 — hosts 파일 수준에서 도메인 차단, 모든 브라우저에 적용
• 키워드 차단기 — 제목에 지정한 단어가 포함된 모든 창을 차단

──────────────────────────────────────
🖥 포커스 런처 — 깊은 작업을 위한 키오스크 모드
──────────────────────────────────────
포커스 런처는 세션이 시작되는 순간 전체 데스크톱을 집중 작업 공간으로 변환합니다:

• Windows 작업 표시줄 숨기기 — "잠깐만 확인하는" 행동을 차단
• 뉴클리어 모드 자동 활성화 — 첫 번째 초부터 최대 차단
• PIN 보호 휴식 — 모든 휴식에 세션 PIN 입력 필요 (충동적 이탈 방지)
• 하드 잠금 모드 — 휴식 없음, 타이머가 끝날 때까지 순수한 깊은 작업
• 충돌 안전 설계 — 앱이 예기치 않게 종료되어도 작업 표시줄이 항상 복원됨

──────────────────────────────────────
⏱ 집중 세션 & 뽀모도로 타이머
──────────────────────────────────────
• 타이머 세션: 15, 25, 45, 60, 90분 — 또는 사용자 지정 시간
• 뽀모도로 모드 — 짧은 휴식과 긴 휴식이 포함된 자동 작업/휴식 사이클
• 세션 PIN — 세션을 일찍 끝낼 수 없도록 자신을 잠금
• 작업 표시줄에 실시간 카운트다운 표시
• 소리 혐오 — 차단된 앱을 열려고 할 때마다 경고음 발생

──────────────────────────────────────
☢ 뉴클리어 모드 — 최강의 강제력
──────────────────────────────────────
탈출 경로를 완전히 차단하고 싶을 때, 뉴클리어 모드는 작업 관리자, PowerShell, 실행 대화 상자, 브라우저 개발자 콘솔 등 30개 이상의 시스템 유틸리티를 차단합니다. 집중하기로 결정했다면, 뉴클리어 모드가 그 결정을 지켜줍니다.

──────────────────────────────────────
📊 통계, 리포트 & 인사이트
──────────────────────────────────────
• 집중 대시보드 — 연속 일수, 완료된 작업, 차단된 앱 시도 횟수, 집중 점수
• 주간 리포트 — 매주 일요일 자동 발송, 피크 시간대와 최고의 날 확인
• 유혹 로그 — 모든 차단 시도를 타임스탬프와 앱 이름과 함께 기록
• 세션 기록 — 모든 세션을 시간과 결과와 함께 로컬에 저장
• 자신의 패턴에 기반한 생산성 트렌드와 추천

──────────────────────────────────────
✅ 작업, 습관 & 노트
──────────────────────────────────────
• 시간대, 소요 시간, 우선순위로 작업 일정 관리
• 반복 작업 — 매일, 평일, 매주
• 연속 완료 기록이 있는 일일 습관 추적기
• 세션 노트와 일일 일지 — 모두 PC에 로컬 저장

──────────────────────────────────────
🔐 100% 개인 정보 보호 — 데이터가 PC를 떠나지 않습니다
──────────────────────────────────────
• 계정 불필요 — 설치 즉시 사용
• 인터넷 연결 불필요 — 모든 기능이 완전히 오프라인으로 작동
• 원격 측정, 분석, 사용 데이터 전송 없음
• 모든 데이터는 %USERPROFILE%\.focusflow\의 SQLite 데이터베이스에 로컬 저장
• 자동 롤링 백업 및 충돌 로그는 로컬에만 저장

──────────────────────────────────────
⚙ 시스템 & 편의 기능
──────────────────────────────────────
• 시스템 트레이에서 조용히 실행 — 마우스 오른쪽 클릭으로 복원, 차단 전환, 종료
• Windows 시작 시 자동 실행 (선택 사항, 관리자 권한 불필요)
• 세션, 차단 이벤트, 주간 리포트에 대한 Windows 알림
• Windows 10 및 Windows 11 지원

──────────────────────────────────────
FocusFlow는 누구를 위한 것인가요?
──────────────────────────────────────
• 공부 세션 중 SNS를 멀리해야 하는 학생
• 깊은 작업이 필요한 개발자, 작가, 디자이너
• 선택적 타이머가 아닌 강력한 약속 도구가 필요한 ADHD 분들
• 다른 집중 앱을 쉽게 무시해버린 경험이 있는 모든 분

"잠깐만 확인하고"라고 말했다가 한 시간을 날린 적이 있다면 — FocusFlow는 당신을 위한 앱입니다.
```
