## 긴 설명
> **Partner Center → Description** (10,000자 이내)

```
FocusFlow는 Windows용 깊은 집중 및 앱 차단 도구로, 주의력을 되찾고 생산적인 습관을 기르며 중요한 작업을 실제로 완료할 수 있도록 도와줍니다.

소셜 미디어, 게임 또는 단순히 자기 책임감 유지에 어려움을 겪든 상관없이, FocusFlow는 부드러운 알림이 아닌 실제 강제 실행을 제공합니다.

──────────────────────────────────────
🔒 진짜 앱 차단
──────────────────────────────────────
FocusFlow는 실행 중인 프로세스를 모니터링하고 차단된 앱이 열리는 순간 닫습니다. 브라우저 탭이 아닌 실제 Windows 애플리케이션입니다. Chrome, Discord, Steam, Spotify 또는 원하는 실행 파일을 차단하세요.

항상 활성, 특정 시간으로 예약, 또는 활성 집중 세션에 연결된 차단을 설정하세요. 일일 허용량을 통해 차단이 자동으로 시작되기 전에 앱별로 제한된 시간을 부여할 수 있습니다.

──────────────────────────────────────
⏱ 집중 세션 & 뽀모도로
──────────────────────────────────────
모든 작업에 대해 시간이 측정되는 집중 세션을 시작하세요. 시간(15, 25, 45, 60 또는 90분)을 선택하거나 사용자 지정 시간을 입력하세요. 뽀모도로 모드를 활성화하면 짧은 휴식과 긴 휴식이 포함된 자동 작업/휴식 사이클이 진행됩니다.

큰 원형 타이머로 세션을 항상 가시적으로 유지하세요. PIN 잠금으로 조기 종료를 방지하면서 일시 정지, 재개 또는 종료할 수 있습니다. 남은 시간을 항상 알 수 있도록 작업 표시줄 제목에 라이브 카운트다운이 표시됩니다.

──────────────────────────────────────
📊 대시보드 & 인사이트
──────────────────────────────────────
매일의 대시보드에는 집중 연속 기록, 완료된 작업, 차단된 앱 시도 횟수, 집중 점수가 표시됩니다. 주간 보고서는 패턴을 요약합니다 — 최고 생산성 시간대, 최고의 날, 평균 세션 길이, 총 집중 시간.

──────────────────────────────────────
✅ 작업, 습관 & 노트
──────────────────────────────────────
시간대, 소요 시간, 우선순위로 작업을 일정에 넣으세요. 반복 작업(매일, 평일, 매주)을 설정하세요. 완료 연속 기록이 있는 일일 습관을 추적하세요. 세션 노트와 일일 일지를 보관하세요 — 모두 로컬에 저장됩니다.

──────────────────────────────────────
🛡 블록 디펜스 & 뉴클리어 모드
──────────────────────────────────────
블록 디펜스는 우회를 방지합니다 — 규칙을 무시하고 싶은 유혹이 있을 때도 강제 실행을 보호합니다. 뉴클리어 모드는 탈출구 없는 최대 집중을 위해 모든 것을 잠급니다.

──────────────────────────────────────
🔐 설계에 의한 개인 정보 보호
──────────────────────────────────────
모든 것이 기기에 남아 있습니다. 계정 불필요. 클라우드 동기화 없음. 데이터 수집 없음. 모든 설정, 작업, 통계는 PC의 SQLite 데이터베이스에 로컬로 저장됩니다.

──────────────────────────────────────
시스템 트레이에서 조용히 실행됩니다. 창이 닫혀도 차단은 활성 상태를 유지합니다. Windows 10 및 Windows 11용으로 구축되었습니다.
```
