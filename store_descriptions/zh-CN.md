## 完整描述
> **Partner Center → Description**（≤ 10,000 个字符）

```
FocusFlow 是一款适用于 Windows 的深度专注与应用拦截工具，帮助您重新掌控注意力、养成高效习惯，并真正完成重要的工作。

无论您是在与社交媒体、游戏还是自我约束作斗争，FocusFlow 都能提供真正的强制执行 — 而不仅仅是温和的提醒。

──────────────────────────────────────
🔒 真正的应用拦截
──────────────────────────────────────
FocusFlow 实时监控运行中的进程，在被屏蔽的应用打开的瞬间将其关闭。不只是浏览器标签 — 而是真实的 Windows 应用程序。屏蔽 Chrome、Discord、Steam、Spotify 或任何您选择的可执行文件。

设置始终开启、按特定时段计划，或与活跃专注会话关联的屏蔽。每日配额允许您在屏蔽自动生效前为每个应用设置有限的使用时间。

──────────────────────────────────────
⏱ 专注会话与番茄钟
──────────────────────────────────────
为任何任务启动计时专注会话。选择您的时长（15、25、45、60 或 90 分钟）或输入自定义时间。启用番茄钟模式，享受自动工作/休息循环，支持短休和长休。

大型圆形计时器让您的会话始终清晰可见。通过 PIN 锁定来防止提前退出，同时可以暂停、继续或结束会话。任务栏标题中显示实时倒计时，让您随时了解剩余时间。

──────────────────────────────────────
📊 仪表盘与洞察
──────────────────────────────────────
您的每日仪表盘显示专注连续天数、已完成任务、被拦截的应用尝试次数和专注评分。每周报告总结您的规律 — 高峰生产时段、最佳工作日、平均会话时长和累计专注时间。

──────────────────────────────────────
✅ 任务、习惯与笔记
──────────────────────────────────────
按时间段、时长和优先级安排任务。设置循环任务（每天、工作日、每周）。追踪每日习惯并记录完成连续天数。保存会话笔记和每日日志 — 全部本地存储。

──────────────────────────────────────
🛡 拦截防护与核弹模式
──────────────────────────────────────
拦截防护防止绕过规则 — 即使您有冲动想要覆盖设置，也能保持强制执行。核弹模式锁定一切，实现无退出通道的最大专注。

──────────────────────────────────────
🔐 隐私设计
──────────────────────────────────────
所有内容都保留在您的设备上。无需账号。无云端同步。不收集数据。所有设置、任务和统计数据均本地存储在您 PC 上的 SQLite 数据库中。

──────────────────────────────────────
在系统托盘中静默运行。即使窗口关闭，拦截也保持活跃状态。专为 Windows 10 和 Windows 11 构建。
```
