## 完整描述
> **Partner Center → Description**（≤ 10,000 个字符）

```
FocusFlow — Windows 深度专注与应用拦截工具

重新掌控你的注意力。FocusFlow 是 Windows 上最强大的专注与生产力工具——它不只是建议你专注，而是强制执行。在进程层面屏蔽干扰应用，将桌面锁定为无干扰工作空间，帮助你建立持久的高效习惯。

无论你是学生、开发者、写作者，还是任何与分心作斗争的人，FocusFlow 都能让你每天发挥出最佳状态。

──────────────────────────────────────
🔒 真正的应用拦截——而非温和提醒
──────────────────────────────────────
FocusFlow 实时监控 Windows 进程，在被屏蔽的应用打开的瞬间将其关闭。不只是浏览器标签——而是真实的应用程序。屏蔽 Chrome、Discord、Steam、Spotify 或任何你选择的可执行文件。

• 按进程名屏蔽任意应用
• 常态屏蔽、定时屏蔽或仅在专注期间屏蔽
• 每日配额——为应用设置时间预算（例如每天 30 分钟 YouTube）
• 网站拦截——在 hosts 文件层面屏蔽域名，适用于所有浏览器
• 关键词拦截——屏蔽标题中含有指定词汇的任何窗口

──────────────────────────────────────
🖥 专注启动器——深度工作信息亭模式
──────────────────────────────────────
专注启动器在会话开始时将整个桌面变成专注工作空间：

• 隐藏 Windows 任务栏——再也不会"顺手看一眼"通知
• 自动激活核弹模式，从第一秒起实现最强拦截
• PIN 码保护休息——每次休息都需要输入会话 PIN 码，防止冲动退出
• 硬锁定模式——零休息，纯粹深度工作直到计时结束
• 崩溃安全设计——即使应用意外退出，任务栏也会自动恢复

──────────────────────────────────────
⏱ 专注会话与番茄钟计时器
──────────────────────────────────────
• 定时会话：15、25、45、60 或 90 分钟——或自定义时长
• 番茄钟模式——自动工作/休息循环，支持短休和长休
• 会话 PIN 码——锁定自己，防止提前结束会话
• 任务栏实时倒计时显示，随时掌握剩余时间
• 声音厌恶——每次尝试打开被屏蔽的应用时触发提示音

──────────────────────────────────────
☢ 核弹模式——最强执行力
──────────────────────────────────────
当你需要零退出通道时，核弹模式屏蔽超过 30 个系统工具——包括任务管理器、PowerShell、运行对话框和浏览器开发者控制台——让你无路可逃。你选择了专注，核弹模式让这个选择成真。

──────────────────────────────────────
📊 数据统计、报告与洞察
──────────────────────────────────────
• 专注仪表盘——每日连续天数、完成任务数、被拦截应用尝试次数、专注评分
• 每周报告——每周日推送，包含你的高峰时段和最佳工作日
• 诱惑日志——每次被拦截的尝试均记录时间戳和应用名称
• 会话历史——每次会话均在本地存储，包含时长和结果
• 基于你自身模式的生产力趋势与建议

──────────────────────────────────────
✅ 任务、习惯与笔记
──────────────────────────────────────
• 按时间段、时长和优先级安排任务
• 循环任务——每天、工作日或每周
• 每日习惯追踪器，带完成连续记录
• 会话笔记和每日日志——全部本地存储在你的电脑上

──────────────────────────────────────
🔐 100% 隐私——数据永不离开你的电脑
──────────────────────────────────────
• 无需账号——安装即用
• 无需网络连接——所有功能完全离线运行
• 不发送任何遥测、分析或使用数据
• 所有数据本地存储于 %USERPROFILE%\.focusflow\ 的 SQLite 数据库中
• 自动滚动备份，崩溃日志仅保存在本地

──────────────────────────────────────
⚙ 系统与便利性
──────────────────────────────────────
• 静默运行于系统托盘——右键可恢复、切换拦截或退出
• 可选开机自启（无需管理员权限）
• Windows 通知——会话、拦截事件及每周报告提醒
• 支持 Windows 10 和 Windows 11

──────────────────────────────────────
FocusFlow 适合谁？
──────────────────────────────────────
• 需要在学习期间远离社交媒体的学生
• 从事深度工作的开发者、写作者和设计师
• 需要硬性约束工具而非可选计时器的 ADHD 人群
• 尝试过其他专注应用却总能轻松忽略的人

如果你曾对自己说"只看一下"然后浪费了一个小时——FocusFlow 就是为你而生的。
```
