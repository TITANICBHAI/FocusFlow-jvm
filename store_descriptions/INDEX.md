## How to add a localized listing in Partner Center

1. In Partner Center, go to **Your app → Store listings**.
2. Click **Add/remove languages** and add the languages above.
3. Open each language listing and paste the **Short description** and **Description** from the matching file.
4. Paste the **Keywords** into the **Search terms** field.
5. Use the same screenshots and logo for all listings (or provide localized screenshots for best results).

## Each file contains

- **Short description** — ≤ 200 characters, SEO-optimized, copy-paste ready
- **Long description** — ≤ 10,000 characters, bullet-pointed, localized, user-friendly
- **Keywords / Search terms** — 7 localized search terms for discoverability
